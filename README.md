# Explore With Me 

Патформа для организации и поиска событий. Пользователи могут создавать, просматривать и участвовать в различных мероприятиях, а администраторы могут управлять контентом и пользователями.

## Технологии и библиотеки

Проект реализован с использованием современных технологий и библиотек:

- **Java 21** — основной язык программирования
- **Spring Boot 3.3.0** — фреймворк для создания веб-приложений
- **Spring Data JPA** — для работы с базой данных
- **Spring Web** — для реализации REST API
- **PostgreSQL** — реляционная база данных
- **Maven** — система управления зависимостями
- **OpenAPI 3.0** — для документирования API
- **Lombok** — для уменьшения шаблонного кода
- **JUnit 5** — для написания тестов
- **Mockito** — для создания моков в тестах

## Технологии

- Java 21
- Spring Boot 3.3.0
- PostgreSQL
- Maven
- OpenAPI 3.0

## Функциональность

### Пользовательские возможности
- Создание и управление событиями
- Участие в событиях
- Просмотр публичных событий и подборок
- Управление своими заявками на участие

### Административные возможности
- Управление пользователями (создание, удаление)
- Управление категориями (создание, редактирование, удаление)
- Управление событиями (модерация, публикация)
- Управление подборками событий

## Архитектура

Проект состоит из двух основных модулей:

1. **main-service** — основной сервис, обрабатывающий бизнес-логику приложения
2. **stats** — сервис статистики для отслеживания просмотров и других метрик

## API

API документировано в соответствии со спецификацией OpenAPI 3.0. Документация доступна в файле `ewm-main-service-spec.json`.

## Запуск

Проект использует Docker для контейнеризации. Для запуска:

1. Убедитесь, что Docker установлен
2. Выполните команду: `docker-compose up`

## Конфигурация

Основная конфигурация находится в `application.yml`. Сервис использует PostgreSQL в качестве базы данных и настроен на работу с локальной базой данных на порту 6542.

## Разработка

Проект использует следующие инструменты качества кода:
- Checkstyle
- SpotBugs
- JaCoCo для измерения покрытия кода тестами
